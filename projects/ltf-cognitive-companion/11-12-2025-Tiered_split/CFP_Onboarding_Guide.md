# CFP Tiered Onboarding Guide

## Tier 1 — Foundations (Model-Agnostic)
- Load CORE Primer
- Initialize USM Baseline
- Enable Universal Modes
- Begin simple continuity workflows

## Tier 2 — Cognitive Alignment (Model-Aware)
- Upload LLM Alignment Bundle
- Activate NISCL
- Apply Model Interpretation Layers
- Validate drift protection
- Confirm mode-behavior consistency

## Tier 3 — Cognitive Orchestration (Multi‑LLM)
- Initialize Multi‑LLM Engine
- Register participating models
- Enable Divergence Detection
- Activate Cross‑Model Knowledge Graph
- Confirm orchestration safety layer
